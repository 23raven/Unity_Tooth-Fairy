# 🧚‍♀️ Tooth Fairy

A small 2D game made in Unity where you play as a Tooth Fairy.

## 🎮 Gameplay

- Collect teeth across the city  
- Each tooth costs **$100**  
- Refill your money at the gold bag  
- Reach the required amount of teeth  
- Return to base before time runs out  

## ✨ Features

- Simple and clean 2D gameplay  
- Timer-based challenge  
- Parallax background  
- UI with live stats (money & teeth)  
- Basic animations and interactions  

## 🛠️ Tech

- Unity (2D)
- C#
- TextMeshPro
